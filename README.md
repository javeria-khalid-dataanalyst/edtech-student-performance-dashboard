# 📊 EdTech Student Performance Dashboard

## Project Overview
An end-to-end Power BI dashboard analyzing 
student performance across an educational An 
institution covering 3 academic years.

## Dataset
- 44,392 assessment records
- 50 students across 4 grade levels
- 8 subjects
- 5 teachers
- 3 academic years (2022-2025)

## Dashboard Pages
| Page | Description |
|------|-------------|
| Academic Overview | Monthly trends, grade distribution, pass rate |
| Subject Intelligence | Subject performance, assessment analysis |
| Student Leaderboard | Top/bottom students, rankings |
| Teacher Performance | Teacher comparison vs benchmark |
| Student Journey | Individual student deep dive |

## Key Findings
- School maintains 78.66% overall pass rate
- Grade B is most common (47.2% of records)
- December scores highest (74.5 avg)
- July scores lowest (67.2 avg) — summer dip
- Louis Dubois, top performer (93.11 avg)
- Projects score highest (73.6) vs Quarter exams (66.0)
- All teachers perform within 2 points of the benchmark

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)

## DAX Measures Written
- Pass Rate %
- Student Rank (ALLSELECTED)
- Avg Score
- Best Score / Worst Score
- Teacher vs Overall Benchmark
- Total Failed / Total Passed
- Perfect Score Count
- YoY Score Change
